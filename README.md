# notnanton.github.io

Personal academic website for Anton O. Wiehe, built with [al-folio](https://github.com/alshedivat/al-folio) and hosted on [GitHub Pages](https://pages.github.com/).

**Live site:** [notnanton.github.io](https://notnanton.github.io)

## Local Development

```bash
bundle install
bundle exec jekyll serve
```

Then open [localhost:4000](http://localhost:4000).
